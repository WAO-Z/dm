# dm

一个纯静态的动漫磁力链接分享页面，直接打开 `index.html` 就能运行。

## 文件说明

- `index.html`：页面结构。这里放真实 DOM 容器、筛选工具栏和卡片模板。
- `styles.css`：页面样式。包含主题变量、响应式布局、卡片动画和兼容性样式。
- `app.js`：页面逻辑。包含数据、筛选、排序、分页、图片懒加载、复制链接和回到顶部。
- `favicon.svg`：浏览器标签页图标。

## 学习重点

- 数据驱动渲染：作品数据集中放在 `animeData`，页面卡片由 JS 根据数据生成。
- DOM 模板复用：`template#animeCardTemplate` 作为卡片骨架，渲染时 clone 出每一张卡片。
- 筛选与排序：搜索、类型、年份和排序条件统一走 `getFilteredAnime()`。
- 性能优化：搜索索引预先生成，磁力链接使用事件委托，封面图使用懒加载。
- 可访问性：下拉框、分页、复制提示和旧浏览器弹窗都带有必要的 aria 属性。
- 响应式布局：CSS 通过媒体查询适配桌面、平板、手机和极窄屏。

## 运行方式

直接双击 `index.html`，或在项目目录启动任意静态服务器后访问页面。
